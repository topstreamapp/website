Топовый чат и классные виджеты для стрима

Чат для вывода на стрим с различными скинами и другие виджеты, БЕСПЛАТНО без ограничений.

Загрузить TopStream 1.0.0>

Для 64-разрядных версий Windows 7/8/8.1/10

Сервисы поддерживаемые виджетом чата

Виджет чата TopStream предназначен для вывода на трансляцию,
и поддерживает все основные стриминговые платформы:

GoodGame!
Пекач
Mixer
WASD.tv
OK Live
Streamlabs
DonationAlerts

Как это выглядит:

Дополнительно, вы получаете следующий функционал:

  • Мультичат для Twitch, YouTube, GoodGame, VK, Peka2, Mixer, который работает в любом стриминговом софте и браузере.
  • Различные скины для чата. Возможность подключать свои скины!
  • Редактор скинов. Создание скинов для чата на стрим!
  • Фильтрация сообщений и ботов.
  • Замена никнеймов пользователей на желаемые.
  • Полная поддержка Twitch, включая все смайлики сервиса, цвета ников, удаление сообщений при банах и все иконки.
  • Поддержка BetterTTV смайлов.
  • Поддержка FrankerFaceZ смайлов.
  • Вывод чата WASD.tv на стрим. Чат с поддержкой WASD.tv.
  • Поддержка GoodGame и Peka2 со смайликами, иконками и цветами ников.
  • Поддержка чатов Mixer.
  • Поддержка всех эмодзи.
  • Возможность открытия любого, даже стороннего виджета в отдельном окне.
  • Возможность встраивать чаты, сайты и виджеты на экран, не создавая окон!
  • Слайдшоу из картинок или профилей из социальных сетей стримера.
  • Универсальный розыгрыш для ваших зрителей.
  • Озвучивание сообщений чата.

Работает на Windows и macOS

Приложение доступно для вашего macOS или Windows устройства. macOS версия временно недоступна.
